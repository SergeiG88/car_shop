### 💼 Сайт автомобильного магазина - Django

```
💻 Команды для запуска приложения:
    
    1. Загрузка данных в базу данных:
    python manage.py import_cars
    
    2. Запуск проекта Flask с помощью команды:
    python manage.py runserver
```
